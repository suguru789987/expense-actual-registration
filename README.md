## 概要

ラクミー経営管理の**費用実績入力機能**(地代家賃実績 等) の独立仕様モック。

オーナー登録フロー機能と同時期に開発されていたため以前は同一リポジトリで管理していたが、独立テーマとしての価値が高いためこちらに切り出し。

## ハイライト

- **地代家賃実績モーダル**: 月選択をプルダウンからテキストラベルに変更する判断
- **月次費用実績の入力フロー**: 費目別(地代家賃/人件費/変動費等)で異なる入力UI
- **検証プラン** (2026-04-10)
- **CS申し送り表 v1→v2** で運用品質を段階強化
- **UIロジック申し送り**: UIの振る舞い詳細を CS向けに翻訳
- **ヘルプページ改訂申し送り**: ドキュメント側との連動
